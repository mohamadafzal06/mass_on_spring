# Spring on Mass Simulation

A simple spring-mass physics simulation built with C and Raylib.

## Description

This project simulates a mass attached to a spring, demonstrating basic harmonic motion and physics principles. Built using the C programming language with the Raylib library for graphics.

## Requirements

- CMake (3.10 or higher)
- C compiler with C99 support
- Raylib library

## Building

```bash
# Create build directory
mkdir build && cd build

# Configure with CMake
cmake ..

# Build the project
make
```

## Running

```bash
./spring
```
